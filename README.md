# 🎮 Tic Tac Toe Web Application

A simple and interactive Tic Tac Toe game built using **HTML, CSS, and JavaScript**.  
The game supports both **Player vs Player (PvP)** and **Player vs Computer (PvC)** modes.

---

## 🚀 Features

- 👥 Player vs Player mode
- 🤖 Player vs Computer mode (basic AI)
- 📊 Score tracking (X, O, Draws)
- 🎯 Win detection with highlighted winning line
- 🔁 Restart game option
- 🔄 Switch between game modes anytime
- 📱 Responsive and clean UI

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

---

## 📂 Project Structure

This project is a **single-file application**, so everything is inside one HTML file:

It contains:
- HTML → structure
- CSS → styling
- JavaScript → game logic

---

## 🎮 How to Run

1. Download or copy the `index.html` file
2. Open it in any modern web browser (Chrome, Edge, Firefox)
3. Start playing instantly — no setup required

---

## 🧠 Game Rules

- Player X always starts first
- Players take turns marking a 3×3 grid
- First player to align 3 marks (horizontal, vertical, or diagonal) wins
- If all cells are filled and no winner → game is a draw
